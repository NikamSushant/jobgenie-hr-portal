# JobGenie HR Portal

Standalone HR portal for discovering candidates from JobGenie AI.

## Setup

1. Clone this repo
2. Copy `.env.example` to `.env` and fill in your values
3. Deploy to Vercel

## Environment Variables

| Variable | Value |
|----------|-------|
| `VITE_SUPABASE_URL` | `https://epustisedanquqfsxidc.supabase.co` |
| `VITE_SUPABASE_ANON_KEY` | Your Supabase anon key |
| `VITE_APP_URL` | Your main JobGenie app URL (for portfolio links) |

## Deploy to Vercel

1. Push this repo to GitHub
2. Import in Vercel
3. Add environment variables in Vercel dashboard
4. Deploy!
